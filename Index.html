Models

Agenda empresarial web para entidades de organización laboral.

Acerca del proyecto

Desarrollar un aplicativo web que ayude a mejorar la comunicación y organización de un empleador administrativo en un entorno sencillo y útil para la organización de las diferentes actividades del día a día, de igual forma hacer uso de la metodología Scrumban para organizar las diferentes actividades del proceso de desarrollo del sistema, además, la base de datos a implementar corresponde a una no relacional, debido a la poca exigencia de dependencia entre las distintas entidades que forman parte del modelo de negocio.

Objetivo específico

  • Establecer el uso de las diferentes tecnologías, frameworks y servicios que mejor se adapten al sistema para obtener un rendimiento óptimo y facilitar el manejo del aplicativo web.
  • Implementar la metodología Scrumban para gestionar de mejor forma el alcance, propósito, actividades y los requisitos del proyecto.
  • Establecer un modelamiento de base de datos tomando como referencia las políticas de negocio para organizar las diferentes dependencias del aplicativo web.

Metodologia

La metodología a usar se basa en la unión de la metodología Scrum y Kanban. La denominada metodologia Scrumban

Roles

Service Request Manager o Product Owner Service Delivery Manager o Flow Master

Arquitectura

La arquitectura de software a implementar es la denominada cliente-servidor.
